# Release Note

## v1.1

### 新增功能
- 托盘菜单新增"打开启动文件夹"选项，方便用户添加开机自启动快捷方式

### Bug 修复
- 修复 Win11 下关闭所有文件夹后 Overlay 仍显示旧列表的问题（竞态条件）

### 优化
- 优化路径显示：移除固定 50 字符截断，改为根据实际可用空间自动截断

## v1.0

### 核心功能
- 标准对话框检测（`#32770` 类名 + 控件特征）
- 文件夹列表收集（通过 `Shell.Application` COM 接口）
- Overlay 窗口显示（WinUI 3，对话框下方外部）
- 导航注入（点击面包屑切换编辑模式 → 设置路径 → 模拟 Enter）
- 防多开（Named Mutex 单实例运行）
- 系统托盘图标（右键菜单配置：显示开关、日志级别、最大项目数、语言）

### UI 功能
- 对话框关闭自动隐藏 Overlay
- 暗色/亮色主题自动适配
- Per-Monitor DPI 适配
- Overlay 圆角显示
- 对话框移动时 Overlay 跟随移动
- Overlay 置顶显示，对话框失去焦点时自动隐藏

### 配置功能
- JSON 配置文件管理
- 托盘菜单直接切换配置（中英文支持）
- 文件夹列表缓存（对话框关闭前缓存，避免重复 COM 调用）
- 导航后跳过刷新（点击文件夹导航后 1 秒内跳过列表刷新）

### 其他
- UAC 提权检测（管理员权限对话框显示 Toast 提示）
- 文件日志（按日期轮转，保留最近 7 天）
